# ToDo App

Simple to-do list application built with HTML, CSS, and vanilla JavaScript.

## Features

- Add a task
- Mark a task as completed
- Delete a task
- Filter tasks by `All`, `Active`, and `Completed`
- Clear all completed tasks

## Project Structure

```text
TodoAPP/
|-- css/
|   `-- style.css
|-- js/
|   |-- app.js
|   |-- storage.js
|   `-- ui.js
`-- index.html
```

## Run Locally

1. Clone the repository.
2. Open `index.html` in your browser.

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## Notes

`storage.js` is present in the project structure and can be used later to add local storage support.
